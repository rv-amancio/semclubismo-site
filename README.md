# Sem Clubismo — Angular Landing Page

Projeto inicial em Angular para a landing page do Sem Clubismo.

## Como rodar

```bash
npm install
npm start
```

Depois acesse:

```bash
http://localhost:4200
```

## Como buildar

```bash
npm run build
```

## Estrutura

- `src/main.ts` — componente standalone principal
- `src/app.html` — estrutura da landing
- `src/app.css` — identidade visual e responsividade
- `src/assets/team` — fotos de Rodrigo, Daniel e Tadeu
